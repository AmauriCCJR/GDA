# GDA - Gestão de Despachos Aduaneiros
![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge) ![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=VERSÃO&message=0.1.9&color=0099ff&style=for-the-badge)

<h2>Repositório para TCC do curso Sistemas para Internet (Tecnólogo) <br /><br>Instituição: Fatec São Roque  - 2024-2026</h2> 


<h3>Atualmente há no mercado uma falta considerável de sistemas no segmento de gestão documental para empresas de pequeno e médio porte voltadas a intermediação de trâmites de despachos aduaneiros que pode acarretar diversos problemas como: a perda documental, custos extras e trabalho repetitivo.  Diante disso o presente repositório apresenta, como proposta de resolução ao problema apresentado, o desenvolvimento da plataforma web de gestão documental GDA (Gestão de Despachos Aduaneiros).</h3> 


## Ferramentas utilizadas
- Javascript
- PHP 
- PostgreSQL
- CodeIgniter 4
